# Academic Portfolio

This repository contains my personal academic portfolio, created as a digital CV
to support my application for undergraduate studies in Software Engineering and
related technical programs at Russian universities.

The portfolio presents my current academic background, technical foundations,
student activities, and future academic plans. It reflects my intention to
reorient my academic path toward a technical specialization and continue my
studies at the master’s level.

This portfolio was prepared specifically for academic and scholarship application
purposes. I sincerely hope it will be considered as part of my application.

- Presentation of technical interests and student activities
